# 阿含经与尼柯耶对照研究计划

这是一个专注于收集、整理 Āgama（汉译阿含）与 Nikāya（巴利尼柯耶）优质多语言译本的项目。旨在通过优化的格式和对照研究，辅助佛法学习者深入研读经藏。

## 项目简介

本项目致力于构建一个高质量的佛经对照阅读环境。目前主要工作集中在文本的数字化整理、格式优化（分章节、排版、脚注）以及不同译本的收集。

## 目前收录

### 汉译阿含 (Āgama)
- **版本**：恒强法师校注版
- **状态**：已收集整理

### 巴利尼柯耶 (Nikāya)
- **版本**：萧式球译本
- **状态**：主体部分已收集完成。
- **备注**：小部（Khuddaka Nikaya）尚有部分经文因未找到合适译本而缺省。

## 格式说明
- 所有文档均已进行格式优化。
- 清晰的章节划分与排版。
- 保留并整理了详细的脚注信息。
