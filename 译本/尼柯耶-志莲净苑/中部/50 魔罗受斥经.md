# 中部. 五十. 魔罗受斥经

这是我所听见的：

有一次，大目犍连尊者住在婆伽的鳄鱼山，毕沙迦逻园的鹿野苑。

这时候，大目犍连尊者在空旷的地方行禅。当大目犍连尊者在行禅的时候，邪恶者魔罗走进他的腹内，钻进他的胃部之中。这时候，大目犍连尊者心想：“为什么我的腹部像装满了豆子那样沉重的呢？”

于是，大目犍连尊者离开行禅的地方，回到他的住处，坐在为他预备好的座位上，然后作出反观。他看见走进自己腹内及钻进自己胃部的邪恶者魔罗，于是对他说：“邪恶者，出来。邪恶者，出来！不要触恼如来和如来的弟子，否则会为你长期带来不利和苦恼！”

这时候，邪恶者魔罗心想：“这个沙门虽然说出这些话，但他是不知道我和看不见我的。即使是他的老师也不能立即知道我，身为弟子的他又怎能知道我呢！”

大目犍连对邪恶者魔罗说：“邪恶者，我知道是你。不要认为我不知道，我知道你就是魔罗！邪恶者，我也知道你这样想：‘这个沙门虽然说出这些话，但他是不知道我和看不见我的。即使是他的老师也不能立即知道我，身为弟子的他又怎能知道我呢！’”

这时候，邪恶者魔罗心想：“这个沙门是知道我和看见我，因此才说出这些话的。”于是魔罗从大目犍连的口中走出来，然后又躲在他的喉头那里。

大目犍连尊者看见躲在自己喉头的邪恶者魔罗，于是对他说：“邪恶者，我看见你在那里，不要认为我看不见，我看见你躲在我的喉头！

“邪恶者，从前我是一个名叫兜尸的魔罗，我有一个名叫迦利的妺妹，你是她的儿子，也就是我的外甥。

“邪恶者，那时候，拘留孙世尊. 阿罗汉. 等正觉出于世上，他有一双上首弟子，名叫聪慧和复生。在说法方面，所有拘留孙世尊的弟子都比不上聪慧尊者，因为他在这方面的特长，人们常以‘聪慧，聪慧’来称赞他，所以他称为聪慧尊者。

“邪恶者，复生尊者常到森林、树下或静室坐禅，他很容易便会得到想受灭尽定正受。有一次，复生尊者坐在树下，进入想受灭尽定。一些路过的牧人和农夫看见了，心想：‘真稀奇！从没有发生过！这位沙门竟能坐化！让我们把他火化吧。’于是，那些牧人和农夫收集一些草、木、牛粪，把复生尊者的身体火化，之后便离去了。在清晨的时候，复生尊者从想受灭尽定正受出定，他打拍一下衣服，便拿着大衣和钵入村化食了。那些替复生尊者火化的牧人和农夫看见他化食，心想：‘真稀奇！从没有发生过！这位坐化的沙门复生了！’因为他在这方面的特长，人们常以‘复生，复生’来称赞他，所以他称为复生尊者。

“邪恶者，这时候，兜尸魔罗心想：‘我不知道这些具戒、质素好的比丘的来处和去处。让我操纵婆罗门居士，使他们责骂、斥骂、触怒、触恼这些比丘吧。当这些比丘受到责骂、斥骂、触怒、触恼的时候，内心可能会改变，这样我便有机会掌控他们了。’

“邪恶者，于是，兜尸魔罗操纵婆罗门居士，使他们责骂、斥骂、触怒、触恼那些具戒、质素好的比丘。婆罗门居士受到兜尸魔罗操纵，这样责骂、斥骂、触怒、触恼那些具戒、质素好的比丘：‘你们这些秃头、卑贱、下劣、低种姓的沙门！说自己在禅修，其实只是双肩下坠、头面低垂、像醉酒那般沈思、纳闷、守候、守望。

“‘就正如捕鼠的猫头鹰，在树上沈思、纳闷、守候、守望。同样地，你们这些秃头、卑贱、下劣、低种姓的沙门！说自己在禅修，其实只是双肩下坠、头面低垂、像醉酒那般沈思、纳闷、守候、守望。

“‘又正如捕鱼的豺，在河岸沈思、纳闷、守候、守望。同样地，你们这些秃头、卑贱、下劣、低种姓的沙门！说自己在禅修，其实只是双肩下坠、头面低垂、像醉酒那般沈思、纳闷、守候、守望。

“‘又正如捕鼠的猫，在巷内的垃圾堆沈思、纳闷、守候、守望。同样地，你们这些秃头、卑贱、下劣、低种姓的沙门！说自己在禅修，其实只是双肩下坠、头面低垂、像醉酒那般沈思、纳闷、守候、守望。

“‘又正如卸下货物的驴子，在巷内的垃圾堆沈思、纳闷、守候、守望。同样地，你们这些秃头、卑贱、下劣、低种姓的沙门！说自己在禅修，其实只是双肩下坠、头面低垂、像醉酒那般沈思、纳闷、守候、守望。’

“邪恶者，那时候，大多数人在身坏命终之后投生在恶趣、地狱之中。

“邪恶者，于是，拘留孙世尊对比丘说：‘比丘们，婆罗门居士受到兜尸魔罗操纵，责骂、斥骂、触怒、触恼具戒、质素好的比丘，希望这些比丘受到责骂、斥骂、触怒、触恼的时候，内心会改变，这样兜尸魔罗便有机会掌控比丘了。

“‘比丘们，来吧，你们要内心带着慈心，向一个方向扩散开去，向四方扩散开去；向上方、下方、横向扩散开去；向每个地方、所有地方、整个世间扩散开去。你们要内心带着慈心，心胸宽阔、广大、不可限量，内心没有怨恨、没有瞋恚。

“‘你们要内心带着悲心……

“‘你们要内心带着喜心……

“‘你们要内心带着舍心，向一个方向扩散开去，向四方扩散开去；向上方、下方、横向扩散开去；向每个地方、所有地方、整个世间扩散开去。你们要内心带着舍心，心胸宽阔、广大、不可限量，内心没有怨恨、没有瞋恚。’

“邪恶者，那些比丘听了拘留孙世尊这样的教导、这样的教法，便到森林、树下和静室，他们内心带着慈心，向一个方向扩散开去，向四方扩散开去；向上方、下方、横向扩散开去；向每个地方、所有地方、整个世间扩散开去。他们内心带着慈心，心胸宽阔、广大、不可限量，内心没有怨恨、没有瞋恚。

“他们内心带着悲心……

“他们内心带着喜心……

“他们内心带着舍心，向一个方向扩散开去，向四方扩散开去；向上方、下方、横向扩散开去；向每个地方、所有地方、整个世间扩散开去。他们内心带着舍心，心胸宽阔、广大、不可限量，内心没有怨恨、没有瞋恚。

“邪恶者，这时候，兜尸魔罗心想：‘我这样做也不知道这些具戒、质素好的比丘的来处和去处。让我操纵婆罗门居士，使他们照料、恭敬、尊重、供养这些比丘吧。当这些比丘受到照料、恭敬、尊重、供养的时候，内心可能会改变，这样我便有机会掌控他们了。’

“邪恶者，于是，兜尸魔罗操纵婆罗门居士，使他们照料、恭敬、尊重、供养那些具戒、质素好的比丘。婆罗门居士受到兜尸魔罗操纵，于是照料、恭敬、尊重、供养那些具戒、质素好的比丘。

“邪恶者，那时候，大多数人在身坏命终之后投生在善趣、天界之中。

“邪恶者，于是，拘留孙世尊对比丘说：‘比丘们，婆罗门居士受到兜尸魔罗操纵，照料、恭敬、尊重、供养具戒、质素好的比丘，希望这些比丘受到照料、恭敬、尊重、供养的时候，内心会改变，这样兜尸魔罗便有机会掌控比丘了。

“‘比丘们，来吧，你们要保持对身体不净随观，对食物生厌离想，对所有世间生不热衷想，对所有行无常随观。’

“邪恶者，那些比丘听了拘留孙世尊这样的教导、这样的教法，便到森林、树下和静室，他们保持对身体不净随观，对食物生厌离想，对所有世间生不热衷想，对所有行无常随观。

“邪恶者，有一次，在上午，拘留孙世尊穿好衣服，拿着大衣和钵入村化食，聪慧尊者在后随从。那时候，兜尸魔罗操纵一个小孩子，使他拿起石子掷破聪慧尊者的头。聪慧尊者的头在流血，于是拘留孙世尊转过身来观看，他说：‘这个兜尸魔罗真是不知尺度！’邪恶者，在拘留孙世尊转过身来观看的时候，兜尸魔罗就在那里命终，投生在大地狱之中。

“邪恶者，那个大地狱有三个名称：六触入处、铁刺、自受。狱卒走来兜尸魔罗那里，对兜尸魔罗说：‘贤者，铁刺将会一枝接一枝地刺到你的心脏，到时你便会知道，这要经历一千年的煎熬。’邪恶者，兜尸魔罗在大地狱之中受铁刺煎熬了许多年、许多百年、许多千年，再在大地狱之中受苦痛感受的煎熬一万年，那时候，兜尸魔罗的身体像人那样，但头部则像鱼那样。

“触恼拘留孙，

与聪慧比丘，

兜尸堕地狱，

受何事煎迫？

百铁刺穿心，

枝枝自领受，

兜尸堕地狱，

受此事煎迫。

拘留孙弟子，

具有无比智；

触恼拘留孙，

与此等弟子，

黑暗者魔罗，

受铁刺之苦。

释迦佛弟子，

具有无比智，

见海中龙宫，

在世一劫长，

美妙琉璃色，

光芒所遍照，

宫女在其中，

作种种舞戏。

触恼目犍连，

黑魔亦受苦。

释迦佛弟子，

具有无比智，

曾受佛吩咐，

息大众吵杂，

以足指动摇，

东园鹿母堂。

触恼目犍连，

黑魔亦受苦。

释迦佛弟子，

具有无比智，

为警策天众，

曾施神通力，

以足指动摇，

天宫胜利殿。

触恼目犍连，

黑魔亦受苦。

释迦佛弟子，

具有无比智，

曾在胜利殿，

问帝释天王，

为我说一遍，

爱尽解脱法，

帝释即为我，

如实详解答。

触恼目犍连，

黑魔亦受苦。

释迦佛弟子，

具有无比智，

曾在净法堂，

梵天面前问，

之前持邪见，

现在能否舍，

于梵世光耀，

能否见无常。

梵天即为我，

如实详解答，

之前持邪见，

现在已能舍，

于梵世光耀，

能洞见无常；

如今已知悉，

无常不恒久。

触恼目犍连，

黑魔亦受苦。

释迦佛弟子，

具有无比智，

具解脱之人，

曾到须弥峰、

东毗提诃洲、

及走遍大地。

触恼目犍连，

黑魔亦受苦。

火不会心想，

要烧愚痴人。

但愚人弄火，

反被火触恼。

魔罗汝亦是，

想触恼如来，

自身反被烧，

如火烧愚者。

触恼如来者，

带来诸罪业，

魔罗汝莫谓，

恶果不会来。

触恼如来者，

长夜积恶业，

是故莫触恼，

如来与比丘。

比丘于园中，

降服此魔罗，

其余恶夜叉，

亦悉皆隐没。”

魔罗受斥经完
